# greatlem0n-base

[![Build container image](https://github.com/isaiahaguilera/greatlem0n-base/actions/workflows/build.yml/badge.svg)](https://github.com/isaiahaguilera/greatlem0n-base/actions/workflows/build.yml)

A custom [bootc](https://github.com/bootc-dev/bootc) container image for my personal development environment.

## Description

This is a customized bootc image based on Universal Blue, tailored for my specific development needs and workflow preferences.

## Usage

To switch to this image from a bootc system:

```bash
sudo bootc switch ghcr.io/isaiahaguilera/greatlem0n-base
```

Then reboot to apply the changes.

## Building Locally

To build this image locally:

```bash
just build
```
